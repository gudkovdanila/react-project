meticulous tourist
==================
Приложение для интерпретации географических, метеорологических и сейсмических данных.
##Описание работы:
Пользователь выбирает точку на карте, после чего отправляются запросы на сайты, предоставляющие метеорологические и сейсмические данные, которые интерпретируются и отображаются на экране.

Отображаемая информация: Погодные условия, (скорость ветра, атмосферное давление, уровень УФ-излучения, уровень загрязненности воздуха, **если найдутся источники**) координаты центров ближайших землетрясений, часовой пояс, местное время.

Получаемая информация группируется по типу данных (погода, экология, сейсмические данные) и отображается на соответствующих вкладках либо выбирается при помощи настроек на общем экране.

На Активном экране основное место занимает карта, предоставленная сервисом Google Maps. Используя предоставляемый Google API происходит определение координат точки, выбранной пользователем. Затем, производится серия запросов на сервисы с публичным API для получения информации, относящейся к данной точке. Получаемые данные интерпретируются и выводятся на экран в соответствии с выбранными настройками.

Пользователь может выбирать получаемые данные, длину радиуса сбора информации, время, для которого информация будет актуальной (прогнозы, исторические данные, текущий момент).
